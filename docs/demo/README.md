Recommended file name:

- `topic-agent-demo.mp4`

Recommended companion assets:

- `topic-agent-demo-cover.png`
- `topic-agent-demo-script.md`

If the repository should not store large video files directly, keep this folder structure and replace the local file reference in the root `README.md` with an external video link.
